# 🎮 Gamer's Paradise - Complete Game Store Platform

A full-stack game store application with user authentication, shopping cart, purchase history, and a professional dashboard. Built with FastAPI (Python backend) and vanilla HTML/CSS/JS (frontend).

## 🚀 Features

### Authentication System
- User registration with password validation (8+ chars, 1 number, 1 symbol)
- Secure login system
- Password generator with copy functionality
- Show/hide password toggle

### 🎮 Game Store
- 5 game categories: Action, RPG, Sports, Racing, Horror
- 25 unique games (5 per category)
- Search functionality across all games
- Add/remove items from shopping cart
- One-click purchase (no payment gateway)

### 📊 User Dashboard
- Profile page with account statistics
- Shopping cart management
- Purchase history tracking
- Total spent calculation
- Real-time cart count updates

## 🛠️ Technology Stack

| Layer | Technology |
|-------|-----------|
| **Backend** | FastAPI (Python) |
| **Frontend** | HTML5, CSS3, JavaScript |
| **Storage** | In-memory dictionaries |
| **Server** | Uvicorn (ASGI server) |
| **Authentication** | Session-based (UUID tokens) |

### Prerequisites
- Python 3.8 or higher
- Modern web browser

# 🔐 Authentication API - Login & Logout System

A complete authentication system with user registration, login, logout, and session management. Built with FastAPI (backend) and vanilla HTML/CSS/JS (frontend).


## 🎯 Features

### 🔐 Authentication
- User registration with validation
- Secure login system
- Session-based authentication
- Logout functionality
- Password strength requirements

### 🔒 Security Features
- Password hashing (bcrypt)
- UUID-based user identification
- Session tokens
- Input sanitization
- CORS protection

### ✨ User Experience
- Real-time password validation
- Show/hide password toggle
- Generate secure password button
- Copy password to clipboard
- Instant feedback messages
