# 🎮 Gamer's Paradise - Complete Game Store Platform

A full-stack game store application with user authentication, shopping cart, purchase history, and a professional dashboard. Built with FastAPI (Python backend) and vanilla HTML/CSS/JS (frontend).

## 🚀 Features

### Authentication System
- User registration with password validation (8+ chars, 1 number, 1 symbol)
- Secure login system
- Password generator with copy functionality
- Show/hide password toggle

### 🎮 Game Store
- 5 game categories: Action, RPG, Sports, Racing, Horror
- 25 unique games (5 per category)
- Search functionality across all games
- Add/remove items from shopping cart
- One-click purchase (no payment gateway)

### 📊 User Dashboard
- Profile page with account statistics
- Shopping cart management
- Purchase history tracking
- Total spent calculation
- Real-time cart count updates

## 🛠️ Technology Stack

| Layer | Technology |
|-------|-----------|
| **Backend** | FastAPI (Python) |
| **Frontend** | HTML5, CSS3, JavaScript |
| **Storage** | In-memory dictionaries |
| **Server** | Uvicorn (ASGI server) |
| **Authentication** | Session-based (UUID tokens) |

### Prerequisites
- Python 3.8 or higher
- Modern web browser

# 🇮🇳 Digital & Technology Industry Analysis — Kolkata and India

A comprehensive competitive analysis project focused on the **Digital Transformation, App Development, Web Development, AI Solutions, and Digital Marketing industries** across **Kolkata and India**.

This project combines:

* Competitive scoring models
* Market opportunity analysis
* Strategic visualization
* Industry segmentation
* Client targeting analysis
* Data-driven business insights using **R**

---

# 📌 Project Objectives

The goal of this project was to:

* Analyze leading technology companies across multiple domains
* Identify major competitors in Kolkata and India
* Compare company capabilities using scoring frameworks
* Detect market gaps and low-competition opportunities
* Visualize industry trends and strategic positioning
* Build management-level analytical reports

---

# 📂 Project Categories

The analysis was divided into the following sectors:

## 1. 📱 App Development Companies

Analysis of:

* Mobile app development firms
* SaaS providers
* Enterprise app companies
* AI-integrated app solutions

### Key Insights

* Enterprise app development is highly competitive
* AI-powered solutions show increasing demand
* Mid-market SaaS opportunities remain underserved
